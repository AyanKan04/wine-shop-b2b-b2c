# Assets Folder

## Mục đích

Folder này chứa các tài nguyên dùng chung cho toàn bộ dự án.

---

# Nội dung lưu trữ

* Logo
* Hình sản phẩm
* Icon
* Background
* Mockup thiết kế
* Tài nguyên UI

---

# Cấu trúc đề xuất

assets/
│
├── images/
├── icons/
├── logos/
└── mockups/

---

# Quy tắc đặt tên

Dùng chữ thường + dấu gạch ngang.

Ví dụ:

* wine-banner.jpg
* admin-icon.png
* login-background.jpg

---

# Quy tắc làm việc

* Nén ảnh trước khi upload
* Không upload file quá nặng nếu không cần
* Không để file trùng lặp
* Sắp xếp đúng folder
