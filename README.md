# wine-shop-b2b-b2c

# Main Branch

## Mục đích

Đây là branch chính và ổn định nhất của dự án Wine Shop B2B/B2C.

Branch này dùng để:

* Lưu phiên bản ổn định của project
* Tổng hợp code từ các branch khác
* Demo và báo cáo đồ án
* Backup chính thức của dự án

---

# Quy tắc làm việc

* Không code trực tiếp trên branch main
* Chỉ merge code đã test ổn định
* Mọi thay đổi phải thông qua Pull Request
* Luôn giữ project chạy ổn định

---

# Nguồn merge chính

Code được merge từ:

* frontend-ui
* backend-api
* database-auth

---

# Workflow

1. Thành viên code trên branch riêng
2. Commit và push
3. Tạo Pull Request
4. Review code
5. Merge vào main

---

# Lưu ý

* Không upload file rác
* Không commit code lỗi lên main
* Kiểm tra conflict trước khi merge
