# wine-shop-b2b-b2c

# Frontend UI Branch

## Mục đích

Branch này dùng để phát triển giao diện người dùng (Frontend UI) cho website bán rượu B2B/B2C.

---

# Thành phần phụ trách

* Trang chủ
* Trang sản phẩm
* Giỏ hàng
* Thanh điều hướng
* Giao diện admin
* Responsive mobile/tablet

---

# Khu vực làm việc chính

frontend/

---

# Quy tắc làm việc

* Chỉ chỉnh sửa code frontend
* Không sửa backend/database nếu không được phân công
* Component cần tái sử dụng hợp lý
* UI cần responsive

---

# Quy tắc đặt tên

## File JS

Dùng camelCase.

Ví dụ:

* loginPage.js
* productService.js

---

## Component React

Dùng PascalCase.

Ví dụ:

* ProductCard.jsx
* AdminSidebar.jsx

---

# Workflow

1. Pull code mới nhất
2. Code giao diện
3. Commit rõ ràng
4. Push branch
5. Tạo Pull Request về main

---

# Ví dụ commit

* Add homepage UI
* Create login form
* Update product card layout
