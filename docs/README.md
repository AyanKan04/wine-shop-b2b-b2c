# Documentation Folder

## Mục đích

Folder này dùng để lưu toàn bộ tài liệu của dự án.

---

# Nội dung lưu trữ

* Phân tích yêu cầu
* Use case
* ERD
* Flowchart
* Tài liệu API
* Báo cáo đồ án
* Meeting note

---

# Cấu trúc đề xuất

docs/
│
├── report/
├── diagrams/
├── api/
└── meeting-notes/

---

# Quy tắc đặt tên

Đặt tên rõ ràng, dễ hiểu.

Ví dụ:

* use_case_diagram.png
* api_documentation.md
* sprint_meeting_01.md

---

# Quy tắc làm việc

* Cập nhật tài liệu thường xuyên
* Không lưu file trùng lặp
* Sắp xếp folder gọn gàng
* Chỉ giữ phiên bản mới nhất nếu không cần backup
